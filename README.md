# VGP-Universe

Y3K Futuristic Digital Lab for VGP Universe.

## Quick Start

```bash
npm install
npm run dev
```

Then open [http://localhost:3000](http://localhost:3000)

## Tech Stack

- Next.js 14+ (App Router)
- TypeScript
- Tailwind CSS
- Framer Motion
- Google Stitch (Design)

## Y3K Design System

- **Cyber Lime**: #BFFF00
- **Electric Cyan**: #00F0FF
- **Obsidian**: #000000

## Structure

```
├── app/
│   ├── layout.tsx      # Root layout with LazyMotion
│   ├── page.tsx        # Main page
│   └── globals.css     # Y3K design tokens
├── components/
│   ├── ui/             # Atomic components
│   └── sections/       # Page sections
├── hooks/              # Custom hooks
└── lib/                # Utilities
```
