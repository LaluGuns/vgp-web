Controlling language: Bahasa Indonesia

## Before enabling creator downloads

1. Print or digitally sign `FLOW_CREATOR_RIGHTS_AND_OPERATOR_AUTHORITY_creator-license-2026-07-21_ID-EN.pdf` in both capacities.
2. Keep the signed PDF and its SHA-256 in an access-controlled evidence folder.
3. Publish the exact canonical ID and EN terms without editing their text.
4. Require an unchecked clickwrap checkbox before each first grant for a track.
5. Store the user/account, track ID, terms ID, terms hash, catalog version, locale, UTC acceptance time, active Pro evidence, and grant ID.
6. Generate a certificate only from a valid stored Grant Record. Never issue the blank template.
7. Keep creator licensing disabled for any track whose ownership, contributors, samples, exclusivity, distributor, or Content ID status later becomes uncertain.
